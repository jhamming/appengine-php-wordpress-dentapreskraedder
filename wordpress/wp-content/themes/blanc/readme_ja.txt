【0. 事前準備】
●カテゴリー
Welcart e-Commerce pluginを使う場合は、Welcartのカテゴリ分けをそのままお使い下さい。

【商品】はWelcartのマニュアルと同様、全ての商品を「商品カテゴリ」以下のカテゴリーに属させ、関連する親カテゴリーにも全てチェックを入れて下さい。
カテゴリーは複数選択することが可能です。

【ブログ】を利用する場合は、特に親カテゴリーなどを作ることなくブログを運営できます。ブログ記事を商品用のカテゴリーに入れないようにして下さい。※Welcartプラグインが有効中であれば、商品用カテゴリーはブログ記事では選択出来ないようになっています。


●no-image画像
ブログのアーカイブページにおいて、通常はアイキャッチ画像またはブログ内の画像からサムネイルが表示されますが、ブログ記事内に画像が存在しない場合、「img」フォルダ内のno-image.jpg(150px x 150px)が代わりに表示されます。こちらも任意の画像に変更することをお勧めします。


【1. トップページ】

●カスタムヘッダーとスライダーの設定方法
1. スライダーの画像の大きさは1600px x 400pxです。
2. ［ダッシュボード］＞［外観］＞［ヘッダー］のページを開く
3. ［画像を選択］から、用意した画像ファイルをアップロード、または既にアップロード済みの場合はメディアライブラリから選択。
4. アップロードした画像が全て、トップページのスライダーに順番に表示されます。
5. ［メディア］＞［ライブラリ］からヘッダーイメージを選び、タイトル・キャプション・説明を編集することにより、画像にタイトル・キャプション・リンクを追加することが出来ます。
6. スライダーから画像を取り除きたい場合は、［メディア］＞［ライブラリ］から画像を［完全に削除する］から削除して下さい。

●新着商品の表示
15日以内に公開された商品が、自動的にランダム表示されます。公開期間を変更したい場合は、front-page.phpを開き、55行目の「strtotime('-15 days')」の数値を変更して下さい。ただしアップデート時には上書きされてしまいますので、子テーマでの変更を推奨します。

●お勧め商品の表示
Welcartカテゴリーの「お勧め商品」にカテゴリー付けされた商品が、4件まで自動的にランダム表示されます。

●フロントページの表示：「最新の投稿」をトップページに選択した場合
ブログ記事のアーカイブが表示されます。

●フロントページの表示：「固定ページ」をフロントページに選択した場合
トップページ用の固定ページに入力した内容（タイトル以外）が、新着商品とお勧め商品の間のスペースに表示されます。バナーや画像を表示したい場合は、トップページ用の固定ページを編集して下さい。

●フロントページの表示：「固定ページ」を投稿用ページに選択した場合
投稿用ページに選択した固定ページが、ブログ記事のトップページとなります。


【2. アーカイブページ】

●サムネイル
アーカイブページに表示されるサムネイルは、1：アイキャッチ画像、2:添付画像、（投稿用記事では　3:no-image.jpg）　の優先順位で表示されます。

●表示件数
1ページ内に表示される件数は、「商品」カテゴリー以下では12件、「ブログ」カテゴリーでは［ダッシュボード］＞［1ページに表示する最大投稿数］から設定した記事数になります。

●タグ
タグを利用することも可能ですが、「商品」カテゴリーに属する記事（商品）とブログ用記事とのタグの共有はなるべく避けて下さい。商品とブログ記事とでタグを共有させると、ブログ用のアーカイブテンプレートが優先適用され、商品一覧ページのような表示になりませんのでご注意下さい。


【3. 商品詳細ページ】

●商品画像
画像の表示幅はおよそ553pxです。それ以下の幅の画像は拡大されて表示されるため、ぼやけてしまいますのでご注意下さい。
商品用画像は1枚以上、必ずご用意下さい。

●レビュー
レビュー表示は、WordPressのコメント機能を使っています。レビューを受け付けるかどうかは、各商品の編集ページの「ディスカッション」ボックスで、「コメントの投稿を許可する。」にチェックを入れるか入れないかに寄ります。
スパム・いたずら防止のため、［設定］＞［ディスカッション］＞［コメント表示条件］で、［コメントの手動承認を必須にする］にチェックを入れておくことをお勧めします。


【4. プラグイン】

このテーマには、最初から「パンくずリスト」「関連商品の表示」「ページャー（ページ送り）」「LightBox」等の機能がつけられています。
これらと同様の機能を持つプラグインの使用は、エラーの原因となる場合がありますのでお避け下さい。

Welcartユーザーの方へ
jQuery Validation Engineの言語は、v1.1.1より自動で取得するようにしております。jQuer Validation Engineで対応言語がない場合は、英語が表示されます。
